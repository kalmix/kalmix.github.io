Retro Commands

Defaulf Prefix: %

Music 🎵

>play (url/name) → Plays your song
>pause → Pause song
>resume → Resume song
>np → Showcase now playing song
>loop → Loops the current playing song
>skip → Skip to the next song
>queue → Showcase the queue
>st → Skip to (queue number)

Info 📚

>help → Shows help

>ping → Shows bot latency

>wiki (Term) → Lookup a term in Wikipedia ES

>covid (Country) → Show COVID-19 Stats

Fun 😁

>comment → "Comment" what you want on YouTube

>di → ​​Make the bot write whatever you want

>trigger → Trigger effect (only available to the author)

>chess → Play chess

>youtube → Watch YouTube with friends

>wanted → "The most wanted in the old west"

>wasted → Wasted from GTA V

>avatar → Shows the user's avatar

>beautiful → Meme "This is beautiful"

>gay → Show avatar with pride flag overlay
