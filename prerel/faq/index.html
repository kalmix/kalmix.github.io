FAQ | Preguntas Frecuentes

¿Qué es internetrepdom?

Somos un movimiento sin fines de lucro que busca mejorar todo lo relacionado con telecomunicaciones en República Dominicana.

¿Cómo se logrará la implementación de fibra óptica?

Mediante recolectas de datos y firmas, para así llevar estos datos a los proveedores de internet

¿Qué es la fibra óptica?

La fibra óptica es la tecnología usada para transmitir información en forma de pulsos de luz mediante hilos de fibra de vidrio o plástico. Esto permite conexiones mucho más rápidas y estables. Para más info accede a Wikipedia.

¿Qué se hará con mis datos?

Los datos serán entregados a los proveedores de internet para que tomen acción. No se venderán o ulitizarán para otro propósito.

¿Para qué necesitan mi número telefónico?

1. Para filtrar entradas de datos ya existentes y para tener base de su veracidad.
2. Para que los respectivos proveedores le puedan contactar en caso de que se instale la fibra.

Nota: No se le contactará para ningún otro propósito que no sea este.
